# 十字光標 Excel 增益集 CF 版



「十字光標」是一個 Excel VBA 增益集，用來在目前選取儲存格所在的行與列顯示醒目的十字高亮，方便在大型表格中快速定位資料。

它主要面向 macOS 版 Excel 使用者，特別是沒有內建 Focus Cell 功能的 Office 版本。普通 `.xlsx`、`.xlsm`、`.xls`、`.csv` 等檔案不需要改成其他格式，只要 Excel 載入增益集即可使用。

## 功能說明

- 選取儲存格時，自動高亮目前儲存格所在的行與列。
- 支援多選範圍，會高亮選取範圍涉及的所有行與列。
- 高亮延伸到目前視窗可見範圍的邊界。
- 作用儲存格本身保持原狀，不套用高亮。
- 遇到合併儲存格時會跳過，不額外覆蓋整個合併區域。
- 內建 8 種純色高亮方案：Green、Blue、Purple、Pink、Orange、Yellow、Red、Gray。
- 支援輸入自訂十六進位色碼，例如 `#20A464`。
- 顏色設定會透過 VBA `SaveSetting` 記住。
- 在 Excel 的「增益集」頁籤提供文字按鈕：啟用、停用、選擇顏色、重置顏色、清理高亮、重建按鈕。
- 停用或關閉時會盡量清理本增益集建立的高亮規則。

## 如何安裝

### 使用既有 `.xlam`

1. 將 `十字光標.xlam` 放到固定位置，例如 `~/Documents/Excel Add-ins/`。
2. 打開 Excel。
3. 選擇 `工具 > Excel 增益集...`。
4. 點擊 `瀏覽...`，選取 `十字光標.xlam`。
5. 勾選「十字光標」增益集。
6. 完整退出 Excel，再重新打開 Excel。

### 從原始碼建立 `.xlam`

1. 打開 Excel，建立一個新的空白活頁簿。
2. 打開 VBA 編輯器。
3. 選擇 `插入 > 模組`，並在屬性視窗把模組名稱改成 `modCrosshair`。
4. 用文字編輯器打開 `vba/Crosshair.bas`。
5. 複製第一行 `Attribute VB_Name = "modCrosshair"` 之後的全部內容，貼到剛建立的 `modCrosshair` 模組。
6. 打開活頁簿的 `ThisWorkbook` 程式碼頁。
7. 用文字編輯器打開 `vba/ThisWorkbook.cls`。
8. 複製 `Option Explicit` 開始到檔案結尾的全部內容，貼到 Excel 的 `ThisWorkbook` 程式碼頁。
9. 在 VBA 編輯器中執行 `偵錯 > 編譯 VBAProject`。
10. 回到 Excel，選擇 `檔案 > 另存新檔...`。
11. 檔案格式選擇 `Excel 增益集 (.xlam)`，檔名建議使用 `十字光標.xlam`。
12. 關閉 VBA 編輯器與目前用來建立增益集的活頁簿。
13. 依照「使用既有 `.xlam`」的步驟載入增益集。

> 第一次建立或替換增益集後，可能會出現十字光標錯亂、重覆等問題。此時只需完整退出 Excel，再重新打開一份 Excel 文檔即可恢復正常。

## 如何使用

載入增益集後，十字光標會預設啟用。切換儲存格時，高亮會自動移動到新的行與列。

在 Excel 的「增益集」頁籤中可以使用以下按鈕：

- `十字光標 啟用`：開啟高亮功能。
- `十字光標 停用`：關閉高亮功能並清理目前高亮。

- `十字光標 選擇顏色`：選擇內建顏色或輸入自訂色碼。

  - 輸入 `1` 到 `8`：使用內建顏色；

  - 輸入內建顏色的名稱，例如 `green`、`blue`、`purple`；

  - 自訂色碼，例如 `#94E6A5`；

- `十字光標 重置顏色`：恢復預設顏色。

- `十字光標 清理高亮`：清理殘留的條件格式與 Shape 光帶。

- `十字光標 重建按鈕`：重新建立「增益集」頁籤中的文字按鈕。

> 如果「增益集」頁籤沒有出現十字光標按鈕，可以從巨集清單手動執行：
>
> - 重建「增益集」頁籤按鈕：`RebuildCrosshairMenu`
> - 移除「增益集」頁籤按鈕：`RemoveCrosshairMenu`
> - 選擇十字光標顏色：`ChooseCrosshairColor`
> - 啟用十字光標：`EnableCrosshair`
> - 停用十字光標：`DisableCrosshair`

## 常見問題

### 為什麼有時看不到增益集按鈕？

Mac Excel 對傳統 VBA `CommandBars` 的支援不完全一致。若按鈕沒有出現，請先從巨集清單執行 `RebuildCrosshairMenu`。若出現重複按鈕，請先執行 `RemoveCrosshairMenu`，再執行 `RebuildCrosshairMenu`。

### 這會修改我的儲存格資料嗎？

不會。增益集只會建立和移除自己的條件格式規則，不會修改儲存格值、公式、工作表或工作簿結構。

### 為什麼我進入 Excel 文件後沒有修改內容，只是點選了幾個儲存格，退出時卻提示是否要儲存變更？

十字光標目前無法像 Excel 原生 Focus Cell 一樣使用純視覺疊加層來顯示高亮。它是透過 VBA 動態建立和移除條件格式規則，讓目前選取儲存格所在的行與列顯示高亮顏色。

這個過程不會修改儲存格的值、公式或原本的資料內容，也會盡量只清理增益集自己建立的高亮規則。不過，因為 Excel 仍然會把條件格式的建立與移除視為工作簿變更，所以關閉文件時可能會提示是否儲存。

通常情況下，如果你只是使用十字光標瀏覽文件，沒有主動修改資料或格式，選擇「不儲存」即可。

### 為什麼受保護工作表沒有高亮？

受保護工作表可能不允許增益集建立或移除條件格式。遇到這種情況時，十字光標會跳過該工作表，避免打斷使用。

### 普通工作簿需要另存為 `.xlam` 嗎？

不需要。`.xlam` 是增益集本身的格式，普通 Excel 檔案保持原格式即可。

## 原始碼結構

- `vba/Crosshair.bas`：十字光標核心邏輯、顏色設定、增益集按鈕。
- `vba/ThisWorkbook.cls`：增益集載入、卸載與 Excel Application 事件綁定。
